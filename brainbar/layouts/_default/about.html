what this is

cmd turns the microsoft acronym soup into a typed shortcut. type a term — mde, e3, foundry, p1 — get a plain-english explainer, the official microsoft definition, rebrand history, the right portal link, and what cert covers it.

no fluff, no gatekeeping, no five paragraphs of seo. just the answer.

why it exists

microsoft has ~700 products and ~200 acronyms that change names every year. mcas became mdca. azure ad became entra id. azure ai studio became azure ai foundry. atp became defender. studying for an exam, writing a sow, or just reading a kb article means constantly googling "what does X mean now?"

cmd is the answer i kept needing for myself. once it existed, it overflowed naturally into something other people could use too — that's the pattern across every tool on a guide to cloud.

who's behind it

built by susanth sutheesh (sush) — microsoft copilot solution engineer at microsoft new zealand. infra admin background. not a developer by training. the same "for people like us" voice that runs through aguidetocloud.com, the youtube channel, and the cert study guides — applied here in url-bar form.

building cmd was selfish. i needed it for myself — every customer call had me re-googling "wait, did mcas become mdca, or was it the other way?" once i'd made the lookup, it cost almost nothing to share. if you're an admin, an se, a student, or just someone who has to write m365 e3 in a doc and wants to know what's actually in it — this is for you. no login. no tracking. no paid tier.

// can't get it to work? manual setup ↓

chrome / edge / brave

  1. settings → search engine → manage
  2. add a new search engine: name: cmd keyword: bb url: https://cmd.aguidetocloud.com/?q=%s
  3. save · type bb <term> in the url bar

firefox

  1. visit this page (firefox auto-detects via opensearch.xml)
  2. right-click the address bar → add cmd
  3. type @bb <term> in the url bar

safari

  1. safari doesn't support custom keywords natively
  2. use the bookmarklet above instead, or install smart keyword search (extension)
