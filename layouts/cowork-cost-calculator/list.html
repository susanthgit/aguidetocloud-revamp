How to Control Your Cowork Burn

  • Set hard caps and alerts in M365 admin center → Copilot → Cost Management. Caps stop spend before it happens; alerts tell you when thresholds hit.
  • Scope to groups, not "All users" in your spending policy. Start with a pilot cohort, measure their patterns for 30 days, then decide whether to expand.
  • Check the Consumption tab weekly — usage by user, group, service. Find the heavy users and ask them what they're getting for the spend.
  • Teach the difference between Copilot (included) and Cowork (metered) — most everyday tasks already work in the $30 seat via Researcher, Analyst, and Office agents. Cowork is for multi-tool orchestration.
  • Use any transition runway before billing starts — if you joined through the Frontier preview, check your tenant's billing start date in the admin center and use the runway to set caps, test patterns, and train users before metered spend begins.

How This Calculator Works

Inputs: You provide the number of Cowork users, average prompts per user per month, and the task complexity mix (light/medium/heavy).

Credit estimates: Based on Microsoft's GA guidance, we model:

  • Light tasks (quick briefs, simple coordination): ~20 credits per prompt (range: 5–50)
  • Medium tasks (multi-step workflows, grounded research): ~100 credits per prompt (range: 50–200)
  • Heavy tasks (long-running, multi-tool, high-context): ~400 credits per prompt (range: 200–1000+)

Billing rate: PayGo = $0.01 per Copilot Credit. If you're using prepaid capacity packs, the per-credit cost is effectively lower, but the estimator shows PayGo for transparency.

Caveats: Actual costs will vary. This is a directional model, not a billing forecast. Your real burn depends on which models users pick, what tools they call, how much context each task needs, and runtime. Microsoft expects per-task costs to fall over time as models get cheaper and Cowork gets better at model-to-task matching.

Sources: Microsoft 365 Blog — Copilot Cowork GA · Usage-based billing overview (Microsoft Learn)
