{{/* blog-prev-next.html — Circular prev/next navigation between blog posts. Uses Hugo's built-in page ordering (by date, newest first). Automatically includes new blogs — no manual config needed. Circular: newest post's "next" links to oldest, oldest post's "prev" links to newest. Usage: {{ partial "blog-prev-next.html" . }} */}} {{ $section := .CurrentSection }} {{ $pages := $section.Pages.ByDate.Reverse }} {{ $count := len $pages }} {{ if gt $count 1 }} {{/* Find current page index */}} {{ $idx := -1 }} {{ range $i, $p := $pages }} {{ if eq $p.RelPermalink $.RelPermalink }} {{ $idx = $i }} {{ end }} {{ end }} {{ if ge $idx 0 }} {{/* Circular: wrap around */}} {{ $prevIdx := cond (eq $idx 0) (sub $count 1) (sub $idx 1) }} {{ $nextIdx := cond (eq $idx (sub $count 1)) 0 (add $idx 1) }} {{ $prev := index $pages $prevIdx }} {{ $next := index $pages $nextIdx }}
← Newer {{ $prev.Title | truncate 45 }} {{ $next.Title | truncate 45 }} Older →
{{ end }} {{ end }}
