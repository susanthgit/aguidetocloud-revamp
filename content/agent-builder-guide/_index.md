faq:
  - question: What is Agent Builder in M365 Copilot?
    answer: Agent Builder is the simplified, no-code experience inside Microsoft 365 Copilot for creating lightweight AI agents. You define instructions, add knowledge sources, and share with your team without any coding.
  - question: Do I need a paid licence to use Agent Builder?
    answer: You can build basic agents with the free Copilot Chat tier (web knowledge only). For work-grounded agents with SharePoint, email, Teams, and file upload, you need either pay-as-you-go billing or a M365 Copilot paid add-on licence.
  - question: What are the knowledge source limits?
    answer: 4 public URLs (max 2 levels deep), up to 100 SharePoint files, 20 uploaded files (512MB max), 5 Teams chats, and all emails (cannot scope to folders). Check the Knowledge Sources tab for full details.
  - question: What happens when I share my agent that reads my emails?
    answer: Each user sees THEIR OWN emails, not yours. The agent runs in the context of whoever is using it. Your email data is never exposed to other users.
  - question: Should I use Agent Builder or Copilot Studio?
    answer: Use Agent Builder for simple Q&A agents grounded in M365 content. Use Copilot Studio if you need multi-step workflows, external APIs, or org-wide deployment. You can copy an agent from Agent Builder to Copilot Studio later.
  - question: Is this tool still being improved?
    answer: Yes! This is V1. Visit our feedback page to suggest improvements or share your Agent Builder experiences.
