faq:
  - question: "Do I need developer skills to use Copilot Studio?"
    answer: "No. Copilot Studio is a low-code platform — you describe your agent in plain language, create topics with a visual editor, and connect to data with pre-built connectors. No coding required for most scenarios."
  - question: "How much does Copilot Studio cost?"
    answer: "Copilot Studio requires a tenant licence that includes 25,000 Copilot Credits per month. Regular messages cost 1 credit, generative AI answers cost 2 credits. Additional credits are available via prepaid packs or pay-as-you-go. Use our Licensing tab to estimate your costs."
  - question: "Can I start with Agent Builder and upgrade to Copilot Studio later?"
    answer: "Yes. Agents built in Agent Builder can be opened in Copilot Studio for more advanced editing. Start simple and upgrade when you need topics, connectors, multi-channel deployment, or advanced orchestration."
  - question: "What's the difference between topics and generative answers?"
    answer: "Topics are structured conversation flows you design — specific paths with branching logic, conditions, and actions. Generative answers are AI-generated responses from your knowledge sources when no topic matches. Most agents use both — topics for critical workflows, generative answers for everything else."
  - question: "Which AI model should I choose for my agent?"
    answer: "Start with the default GPT-4.1 mini — it's fast and cost-effective for most Q&A scenarios. Upgrade to GPT-4.1 or ChatGPT-5 for complex content creation, or Claude/GPT-5 Reasoning for deep analysis tasks. Use our Features & Limits tab to compare all available models."
  - question: "What's the difference between child agents and connected agents?"
    answer: "Child agents are lightweight subagents within your main agent — they share context and are managed together. Connected agents are fully independent agents with their own settings, model, and deployment. Use child agents for logical grouping within one solution; use connected agents when different teams manage different agents independently."
  - question: "Can my agent call external APIs and systems?"
    answer: "Yes. Copilot Studio has 1,000+ connectors for services like ServiceNow, Salesforce, SAP, and any REST API via custom connectors. You can also trigger Power Automate cloud flows from conversations for complex workflows."
