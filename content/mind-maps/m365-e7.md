faq:
  - question: "What's in E7 that I can't get in E5 + Copilot?"
    answer: "Two things. (1) The FULL Entra Suite — not just Entra ID P2, but Internet Access, Private Access (the ZTNA / VPN replacement), full ID Governance, and Verified ID. Bought separately the Entra Suite costs ~$12/user/month. (2) Agent 365 — the AI agent governance platform with Entra Agent ID, Purview for Agents, and Defender for Agents. Agent 365 is currently only sold via E7 (no separate SKU at GA). If you want Agent 365 today, E7 is the path."
  - question: "Is E7 worth it vs E5 + Copilot add-on?"
    answer: "The maths: E5 ($60) + Copilot add-on ($30) + Entra Suite ($12) = $102/user/month. E7 = $99/user/month list. So you save $3/user + get Agent 365 included. Bigger savings during launch promotions (until Dec 31, 2026). E7 makes sense IF you're already buying or planning to buy Copilot AND Entra Suite. If you only need E5 + Copilot, the add-on path is fine. If you have many makers building agents, Agent 365 alone may justify E7."
  - question: "When can I actually buy E7?"
    answer: "E7 became orderable in May 2026. Microsoft's launch promotions (~Dec 31, 2026) include discounted upgrades from E5 + Copilot bundles and prepaid commit incentives. As of May 2026, E7 is enterprise (300+ seats) only — no Business SKU equivalent at this tier. Education and Government variants follow a few weeks behind enterprise."
  - question: "Do I need E7 if I already have E5 + Copilot?"
    answer: "Not yet, unless you specifically want Agent 365 or the full Entra Suite. The two genuine reasons to upgrade: (1) you have makers building custom agents and need governance NOW, (2) you're planning ZTNA / VPN replacement and would buy Entra Private Access anyway. If neither applies, stay on E5 + Copilot and watch how Agent 365 evolves through 2026."
