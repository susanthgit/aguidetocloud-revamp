faq:
  - question: "What's the difference between Azure Pass and Azure free subscription?"
    answer: "Azure Pass is a sponsored credit (usually $100-$200) Microsoft hands out at training events, certification bootcamps, hackathons, and partner programs. The standard Azure free subscription is $200 + 12 months free — anyone can sign up. Pass advantages: doesn't require a credit card (good for students who don't have one). Disadvantages: shorter activation window (must be redeemed within 90 days), expires faster (usually 30 days)."
  - question: "Do I need a credit card for Azure Pass?"
    answer: "No — that's the main reason students and training attendees use it. Azure Pass is identity-verified through the access code you receive at the event, not through a credit card. Once the Pass credit runs out (or expires), the subscription automatically ends — Microsoft won't charge you anything because there's no payment method on file."
  - question: "How do I claim my Azure Pass code?"
    answer: "Go to [microsoftazurepass.com](https://www.microsoftazurepass.com/), sign in with a Microsoft account (use a personal Outlook / Hotmail account, not a work account tied to an existing tenant), then enter your promotional code. Activation takes a few minutes. The full process is shown in the video — start to finish in about 5 minutes."
  - question: "My Azure Pass code isn't working — what's wrong?"
    answer: "Three common causes: (1) you already redeemed an Azure Pass in the past (one per Microsoft account, lifetime); (2) the code expired (most are 90-day windows from issuance); (3) you signed in with a work account already linked to an Azure tenant. Try a fresh personal Microsoft account, and double-check the issue date of the code. If still stuck, the event organiser usually has a backup channel to reissue codes."
  - question: "Can I use Azure Pass for certification labs and exam prep?"
    answer: "Yes — that's exactly what it's designed for. The $100-$200 credit comfortably covers a few weeks of [AZ-104](/cert-tracker/az-104/) / AZ-204 lab work if you stay on B-series VMs and basic storage. Spin up resources, do the exercise, delete them — that's the rhythm. Microsoft's [free Azure hands-on labs](https://learn.microsoft.com/en-us/training/azure/) and the [official AZ-104 GitHub labs](https://aka.ms/az104labs) work well with Pass credit."
