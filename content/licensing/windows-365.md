## Frequently Asked Questions

**1. Do I still need a Windows licence?**

Windows 365 includes the Windows licence. But you still need an M365 plan (E3/E5/Business) for Office apps and identity (Entra/Intune).

**2. Can users access their Cloud PC offline?**

No — Windows 365 requires an internet connection. It's streamed, not installed locally.
