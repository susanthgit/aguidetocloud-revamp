## What's in the Purview Suite?

| Module | What It Does |
|--------|-------------|
| **Advanced DLP** | Endpoint DLP, exact data match, trainable classifiers |
| **eDiscovery Premium** | Machine learning, predictive coding, custodian management |
| **Insider Risk Management** | Detect data theft, policy violations, risky behaviour |
| **Communication Compliance** | Monitor for code of conduct violations |
| **Information Barriers** | Segregate user groups for regulatory compliance |
| **Advanced Audit** | Up to 10-year log retention |
| **Records Management** | Automated retention, file plans, regulatory records |

> **💡 E3 + Purview vs E5:** E3 ($39) + Purview ($12) = $51. E5 ($60) includes Purview features PLUS security (Defender P2, Entra P2) and Teams Phone. E5 is usually better value unless you specifically need compliance without security.

## Frequently Asked Questions

**1. Who needs the Purview Suite?**

Regulated industries — finance, healthcare, legal, government. If you face audit requirements, data retention mandates, or insider threat risks, and you're on E3, this is your compliance upgrade.
