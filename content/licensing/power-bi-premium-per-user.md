## Pro ($10) vs Premium per User ($20)

| Feature | Pro ($10) | Premium/User ($20) |
|---------|:---------:|:------------------:|
| Reports and dashboards | ✅ | ✅ |
| Scheduled refresh | 8x/day | 48x/day |
| Dataset size | 1 GB | **100 GB** |
| **Paginated reports** | ❌ | ✅ |
| **AI features (AutoML)** | ❌ | ✅ |
| **Deployment pipelines** | ❌ | ✅ |
| **Dataflows Gen2** | ❌ | ✅ |

> **💡 Tip:** PPU is the per-user alternative to Power BI Premium capacity ($4,995+/month). If you have fewer than ~250 Power BI users, PPU is usually cheaper than capacity licensing.

## Frequently Asked Questions

**1. Can PPU users share with Pro users?**

PPU content can only be viewed by other PPU users (or Premium capacity users). Pro users cannot view PPU workspace content. Plan your licencing accordingly.
