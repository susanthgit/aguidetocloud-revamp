## EMS E3 vs E5

| Feature | EMS E3 ($12) | EMS E5 ($18) |
|---------|:------------:|:------------:|
| Intune P1 | ✅ | ✅ |
| Entra ID P1 (Conditional Access) | ✅ | ✅ |
| Azure Information Protection P1 | ✅ | ✅ |
| **Entra ID P2 (PIM, risk-based)** | ❌ | ✅ |
| **Defender for Cloud Apps (CASB)** | ❌ | ✅ |
| **AIP P2 (auto-classification)** | ❌ | ✅ |

> **💡 EMS is a "bridge" add-on** for organisations on Office 365 plans who need identity and device management. If you're buying new, go with Microsoft 365 E3/E5 which includes EMS.

## Frequently Asked Questions

**1. Is EMS being retired?**

Not officially, but Microsoft is clearly positioning M365 E3/E5 (which bundle EMS) as the default. EMS remains available for specific scenarios.
