## P1 vs P2

| Feature | P1 (in E3) | P2 ($9 add-on) |
|---------|:----------:|:--------------:|
| Conditional Access | ✅ | ✅ |
| MFA, SSO | ✅ | ✅ |
| Self-service password reset | ✅ | ✅ |
| **PIM (just-in-time admin)** | ❌ | ✅ |
| **Identity Protection (risk scoring)** | ❌ | ✅ |
| **Access reviews** | ❌ | ✅ |

> **💡 PIM in plain English:** Instead of having permanent Global Admins, PIM makes admins "request" elevated access for a limited time. If no one requests it, no one has it. This massively reduces your attack surface.

## Frequently Asked Questions

**1. Is Entra P2 included in E5?**

Yes — M365 E5 includes Entra ID P2. This standalone $9 add-on is for E3 customers who need PIM and Identity Protection without upgrading to E5.
