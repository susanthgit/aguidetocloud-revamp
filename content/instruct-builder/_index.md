faq:
  - question: "How does the Agent Instruction Builder work?"
    answer: "Answer 5 simple questions — pick your platform, describe what the agent does, choose a tone, set boundaries, and specify knowledge sources. The tool instantly generates polished, platform-optimised instructions with all best practices baked in. You can also start from one of 10 pre-built templates."
  - question: "Which platforms are supported?"
    answer: "M365 Agent Builder (declarative agents), Copilot Studio, ChatGPT Custom GPTs, Claude Projects, and OpenAI Assistants API. Each platform has different formatting rules and character limits — the tool handles all of that automatically."
  - question: "What makes good agent instructions?"
    answer: "Good instructions are specific, structured, and include clear boundaries. They define the agent's role, tone, what it can and can't help with, and how to handle questions it can't answer. The best instructions use positive redirects ('Instead, try...') rather than just saying 'Don't'. This tool applies all these best practices for you."
  - question: "How long should agent instructions be?"
    answer: "It depends on the platform. M365 Agent Builder and Copilot Studio allow up to 8,000 characters. ChatGPT GPT Builder also allows 8,000 characters. The tool shows a live character count and warns you if you exceed the limit. Generally, shorter and more focused instructions perform better than long ones."
  - question: "Is my data safe?"
    answer: "100% safe. Everything runs entirely in your browser — no data is sent to any server, no API calls are made, and nothing is stored. Your agent instructions never leave your device."
  - question: "Is this tool still being improved?"
    answer: "Yes! We regularly update the templates, platform configurations, and best practice patterns as platforms evolve. [Share your feedback](/feedback/) to help us improve."
