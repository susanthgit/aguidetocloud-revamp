## About the DATABRICKS-ML-ASSOCIATE Exam

> Master the Databricks Certified Machine Learning Associate exam (1 March 2025 guide) — the Databricks Machine Learning platform (ML Runtime, AutoML, Feature Engineering in Unity Catalog, MLflow tracking and the Unity Catalog model registry), data processing and feature engineering, model development with Spark ML, scikit-learn and Hyperopt, and model deployment for batch, streaming and real-time inference.

The complete practice exam for the Databricks Certified Machine Learning Associate certification (current 1 March 2025 exam guide). Covers the Databricks Machine Learning platform (MLOps best practices, ML Runtime, AutoML for model and feature selection, Feature Engineering and feature store tables in Unity Catalog with the FeatureEngineeringClient, online vs offline feature tables, MLflow Tracking and the MLflow Client API, registering and tagging models in the Unity Catalog model registry, promoting a challenger model to champion with aliases, and promoting code vs models), data processing (summary statistics on Spark DataFrames, outlier removal with standard deviation or IQR, visualizing and comparing categorical and continuous features, imputing missing values with mean/median/mode, one-hot encoding and when it is appropriate, and log-scale transformation), model development (algorithm selection, mitigating class imbalance, Spark ML estimators vs transformers and training pipelines, hyperparameter tuning with Hyperopt fmin and random/grid/Bayesian search, parallelizing single-node tuning with SparkTrials, cross-validation vs train-validation split, counting models in a grid-search plus cross-validation, classification metrics F1/Log Loss/ROC-AUC, regression metrics RMSE/MAE/R-squared, choosing the right metric, exponentiating log-transformed targets, and the bias-variance tradeoff), and model deployment (batch, streaming, and real-time serving approaches, deploying a custom model to a Mosaic AI Model Serving endpoint, pandas and Spark UDF batch inference, streaming inference with Delta Live Tables, querying a real-time endpoint, and splitting traffic between served model versions) — every question a real-world scenario with full explanations.

## Who Should Take This Exam?

The Databricks Certified Machine Learning Associate is designed for **data scientists and machine learning practitioners** who build, tune, and deploy models on the Databricks Data Intelligence Platform. It validates practical, entry-level machine-learning skills across AutoML, Feature Engineering in Unity Catalog, MLflow tracking and the model registry, model development with Spark ML and scikit-learn, and batch, streaming, and real-time model deployment. All machine-learning code on the exam is in Python.

**Prerequisites:** None (course attendance + 6 months hands-on machine learning experience on Databricks recommended)

**Typical study time:** 4-8 weeks of focused study

## Exam Domains & Weights

The DATABRICKS-ML-ASSOCIATE exam covers **4 domains**. Focus your study time based on the weights below — higher-weighted domains have more exam questions.

| Domain | Weight | Practice Qs |
|--------|--------|-------------|
| Databricks Machine Learning | 38% | 95 |
| Data Processing | 19% | 47 |
| Model Development | 31% | 78 |
| Model Deployment | 12% | 30 |
| **Total** | **100%** | **250** |

> 💡 **Study tip:** Databricks Machine Learning (38%) and Model Development (31%) together are more than two-thirds of the exam. Get fluent with AutoML, Feature Engineering in Unity Catalog (the FeatureEngineeringClient `create_table` workflow), MLflow tracking, and the Unity Catalog model registry's aliases (`@champion`/`@challenger`) — and with Spark ML pipelines, Hyperopt `fmin` tuning, cross-validation, and choosing the right metric (AUC and F1 for imbalanced classes, RMSE vs MAE, and remembering R-squared can be negative). Data Processing (19%) rewards hands-on imputation, one-hot encoding (and when to avoid it for high-cardinality or tree models), and log transforms. Model Deployment (12%) is the smallest domain but tests batch vs real-time vs streaming serving, Mosaic AI Model Serving endpoints, and streaming inference with Delta Live Tables.
