## About the AT0-001 Exam

> Vendor-neutral IT automation — scripting, source control, IaC, config management, CI/CD

Covers all 4 domains: Automation Coding Concepts, System Configuration, Continuous Integration, and Continuous Delivery. Master automation scripting (Bash/PowerShell/Python), Git source control, Infrastructure as Code (Terraform/CloudFormation/Bicep), configuration management (Ansible/Puppet/Chef), RESTful CRUD, secrets and artifact management, CI pipelines (Jenkins/GitLab CI/GitHub Actions), and continuous-delivery strategies (canary/blue-green/rolling).

## Who Should Take This Exam?

The AT0-001 (CompTIA AutoOps+) is designed for **automation, DevOps, cloud, and systems professionals** who want to validate their ability to automate infrastructure, manage CI/CD pipelines, and support modern operations. It is part of CompTIA's **Expansion Series** — it builds on the core certification stack rather than replacing it.

**Recommended experience:** 2-3 years in core IT operations (network, cloud, or systems administration). CompTIA recommends earning **Network+, Linux+, Cloud+, and Server+** first, though none is a hard prerequisite for sitting the exam.

**Typical study time:** 4-8 weeks of focused study
