## About the AWS-AIP-C01 Exam

> Master the AWS Certified Generative AI Developer - Professional exam (AIP-C01) — building production generative-AI applications on AWS with Amazon Bedrock foundation models, Knowledge Bases and RAG, Bedrock Agents and AgentCore, Guardrails, prompt engineering, agentic AI with Strands and the Model Context Protocol (MCP), safety and governance, cost and performance optimization, and evaluating and troubleshooting GenAI systems in production.

The complete practice exam for the AWS Certified Generative AI Developer - Professional certification (exam AIP-C01). It covers Foundation Model Integration, Data Management and Compliance (designing GenAI solutions, selecting and configuring foundation models, Amazon Bedrock Cross-Region Inference and Provisioned Throughput, data validation and multimodal pipelines, vector stores with Amazon Bedrock Knowledge Bases / OpenSearch / Aurora pgvector, Amazon Titan and Nova embeddings, chunking and retrieval, reranking, and prompt engineering and governance with Bedrock Prompt Management and Prompt Flows), Implementation and Integration (agentic AI with Strands Agents, AWS Agent Squad and MCP, Amazon Bedrock Agents and AgentCore, model deployment strategies, the Converse API, enterprise integration with API Gateway / EventBridge / Step Functions, and CI/CD and GenAI-gateway patterns), AI Safety, Security and Governance (Amazon Bedrock Guardrails content filters, denied topics, sensitive-information filters, contextual grounding and Automated Reasoning checks, prompt-injection defense, VPC endpoints and PrivateLink, PII detection with Comprehend and Macie, model cards, CloudTrail auditing, and responsible-AI practices), Operational Efficiency and Optimization (token efficiency, prompt caching, tiered model selection, latency-optimized inference, auto-scaling, and observability with CloudWatch and Bedrock Model Invocation Logs), and Testing, Validation and Troubleshooting (Amazon Bedrock Model Evaluation, LLM-as-a-judge and RAG evaluation, A/B and canary testing, and diagnosing context-window, retrieval and prompt issues) — every question a real-world AWS generative-AI scenario with full explanations.

## Who Should Take This Exam?

The AWS Certified Generative AI Developer - Professional (AIP-C01) is designed for **GenAI developers, ML engineers, application developers, and AI solutions architects** who build, integrate, secure, optimize, and troubleshoot production generative-AI applications on AWS with Amazon Bedrock and Amazon SageMaker AI. It validates hands-on GenAI development skills, so **practical experience building production applications on AWS plus one year of hands-on experience implementing GenAI solutions (RAG, agents, foundation-model integration) is strongly recommended**. It builds naturally on the AWS Certified AI Practitioner and the AWS Machine Learning certifications.

**Prerequisites:** None (2+ years building production applications on AWS and 1 year of hands-on GenAI experience are recommended)

**Typical study time:** 8-12 weeks of focused study
