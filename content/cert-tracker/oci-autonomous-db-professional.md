## About the OCI-AUTONOMOUS-DB-PROFESSIONAL Exam

> Master the Oracle Autonomous AI Database Professional exam (1Z0-931-26) — provisioning and managing Autonomous Database Serverless and Dedicated on Exadata, ECPU scaling and lifecycle, the Dedicated Exadata Infrastructure → VM Cluster → Autonomous Container Database → Autonomous Database hierarchy, performance tuning with Automatic Indexing and Performance Hub, security with Oracle Data Safe, TDE and customer-managed keys, the integrated Database Actions tools, Select AI natural-language-to-SQL, AI Vector Search, MCP server integration for AI agents, converged-data and JSON Relational Duality Views, and migration with the Pre-migration Advisor, Data Pump, Zero Downtime Migration and GoldenGate.

The complete practice exam for the Oracle Autonomous AI Database Professional certification (exam 1Z0-931-26, the current 2026 version that replaces the retired 1Z0-931-25). It covers Implementing Autonomous Database Fundamentals (the self-driving, self-securing and self-repairing architecture on Exadata, ATP versus ADW workload types, ECPU versus OCPU compute, secure public and private endpoints with ACLs, auto-scaling and lifecycle), Applying Autonomous Database Serverless Architectures (the shared-Exadata model, full and refreshable clones, Autonomous Data Guard, automatic backups and Point-in-Time Restore), Implementing Autonomous Database Dedicated Deployments (the Exadata Infrastructure, Autonomous Exadata VM Cluster, Autonomous Container Database and Autonomous Database hierarchy, Exadata Cloud@Customer, maintenance windows and IAM fleet policies), Managing, Monitoring and Optimizing Performance (Performance Hub with AWR, ASH and SQL Monitor, Automatic Indexing, consumer-group services, Oracle Data Safe, TDE and OCI Vault customer-managed keys, and REST and CLI automation), Utilizing Autonomous Database Tools (Database Actions, Oracle APEX, Oracle Machine Learning, Data Studio and Graph Studio), Applying Select AI (the SELECT AI runsql, showsql, narrate, explainsql and chat actions, AI profiles and supported LLM providers), Implementing MCP Server Integration (exposing the database as a Model Context Protocol server over ORDS for AI agents), Designing Unified Data Management and AI Analytics (the converged database, JSON Relational Duality Views and AI Vector Search with HNSW and IVF indexes and VECTOR_DISTANCE), and Understanding Migration and Data Integration (the Pre-migration Advisor, Data Pump with DBMS_CLOUD, Zero Downtime Migration and GoldenGate) — every question a real-world Autonomous Database scenario with full explanations.

## Who Should Take This Exam?

The Oracle Autonomous AI Database Professional (1Z0-931-26) is designed for **Oracle DBAs, cloud database engineers, and data architects** who provision, secure, scale, monitor, optimize, and migrate Oracle Autonomous Database — across the Serverless and Dedicated deployment models — and who put its in-database AI features (Select AI, AI Vector Search, and MCP server integration) to work. It validates professional-level Autonomous Database skills, so **hands-on experience with Autonomous Database provisioning, Database Actions, and the DBMS_CLOUD / DBMS_CLOUD_AI packages is strongly recommended**. It builds naturally on the OCI Foundations and Architect Associate certifications.

**Prerequisites:** None required (Oracle recommends hands-on Autonomous Database experience; the free Oracle Autonomous AI Database learning path is a good starting point)

**Typical study time:** 6-10 weeks of focused study

## Exam Domains & Weights

The OCI-AUTONOMOUS-DB-PROFESSIONAL exam covers **9 domains**. Focus your study time based on the weights below — higher-weighted domains have more exam questions.

| Domain | Weight | Practice Qs |

> 💡 **Study tip:** **Autonomous Database Fundamentals** and **Dedicated Deployments** carry the most weight (18% each) — start there, and be sure you can cleanly distinguish Serverless from Dedicated (tenancy isolation, not speed) and the ADB-D four-level hierarchy. The AI domains (Select AI, MCP, and AI Vector Search under Unified Data Management) are smaller but distinctive and heavily favored on the current exam — know the SELECT AI actions and that AI Vector Search uses VECTOR_DISTANCE, not pgvector operators.
