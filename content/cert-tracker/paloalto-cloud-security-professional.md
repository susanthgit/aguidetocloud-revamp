## About the CloudSec-Pro Exam

> Master Cortex Cloud (formerly Prisma Cloud) CNAPP — cloud posture, cloud runtime, application security, and Cortex/SOC fundamentals.

The Palo Alto Networks Certified Cloud Security Professional (CloudSec-Pro) validates the skills needed to secure multi-cloud environments with **Cortex Cloud**, Palo Alto's unified Cloud-Native Application Protection Platform (CNAPP) — the evolution of Prisma Cloud. It covers five domains: Security Operations Center (SOC) Fundamentals, Cortex Fundamentals, Cloud Posture Security (CSPM, KSPM, AI-SPM, DSPM, agentless scanning, identity, compliance), Cloud Runtime Security (Cloud Workload Protection, Cloud Detection and Response, Web Application and API Security), and Application Security (ASPM, CI/CD posture, SCA, Infrastructure as Code security, secrets scanning). Every practice question is an original Cortex Cloud scenario across AWS, Azure, GCP, and OCI with detailed explanations and exam tips.

## Who Should Take This Exam?

The CloudSec-Pro is a **professional-level** certification for cloud security engineers, SOC analysts, DevSecOps engineers, and cloud security architects who design, deploy, and operate CNAPP controls on Cortex Cloud (or Prisma Cloud). Hands-on experience securing cloud workloads, Kubernetes, and CI/CD pipelines is recommended.

**Typical study time:** 6-10 weeks of focused study

## Palo Alto Certification Path

Palo Alto's role-based program runs Apprentice → Analyst → Professional. The CloudSec-Pro is the **cloud security** Professional certification — distinct from the network-security (NGFW) track. It replaces the retired PCCSE (Prisma Certified Cloud Security Engineer).
