## About the SSE Engineer Exam

> Master Prisma Access and Prisma SASE — plan, deploy, secure, operate, and troubleshoot Palo Alto Networks' cloud-delivered Security Service Edge.

The Palo Alto Networks Certified Security Service Edge (SSE) Engineer certification validates the skills needed to deploy and run **Prisma Access** — Palo Alto's cloud-delivered SSE/SASE platform for mobile users, remote networks, and private applications. It covers four domains: Prisma Access Planning and Deployment (architecture, security processing nodes, routing, mobile users and Explicit Proxy, remote networks, service connections, Colo-Connect and the ZTNA Connector, the Cloud Identity Engine, and the Prisma Browser), Prisma Access Services (App Acceleration, IoT Security, Privileged Remote Access, SaaS Security, Enterprise DLP, AI Access Security, Remote Browser Isolation, Security/Decryption/QoS policy, security profiles, and User-ID), Prisma Access Administration and Operation (managing with Panorama and Strata Cloud Manager, the Strata Logging Service, and Best Practice Assessment), and Prisma Access Troubleshooting (Activity Insights, the Command Center, ADEM, and connectivity plus traffic-enforcement troubleshooting). Every practice question is an original SSE engineering scenario with detailed explanations and exam tips.

## Who Should Take This Exam?

The SSE Engineer is a **professional-level** certification for SSE/SASE engineers, network security engineers, and professional-services consultants who deploy, operate, and troubleshoot Prisma Access. Hands-on experience with Prisma Access mobile users, remote networks, security policy, and the Strata Cloud Manager or Panorama management models is recommended.

**Typical study time:** 6-10 weeks of focused study
