## About the SnowPro Advanced: Data Engineer Exam

> Master the Snowflake SnowPro Advanced: Data Engineer certification (DEA-C02) — data movement and ingestion pipelines, performance optimization, storage and data protection, data governance, and data transformation.

The complete practice exam for the Snowflake SnowPro Advanced: Data Engineer certification (DEA-C02). This advanced, role-based exam goes deeper than SnowPro Core — it tests how you build and operate production data pipelines on Snowflake and choose the right capability for each trade-off. Covers data movement (bulk COPY INTO and stages on S3 / Azure / GCS, file formats and INFER_SCHEMA, load troubleshooting, Snowpipe and Snowpipe Streaming, the Kafka / Spark / Python connectors, streams and CDC, Openflow, continuous pipelines, Secure Data Sharing and listings, Streamlit apps, external / Iceberg / hybrid tables, the Horizon Catalog, schema evolution, and unloading), performance optimization (warehouse sizing and multi-cluster scaling, Snowpark-optimized warehouses, the Query Acceleration and Search Optimization services, clustering keys and micro-partition pruning, materialized views, caching, Query Profile, resource monitors and cost control, and continuous-pipeline monitoring with task history, alerts, and data metric functions), storage and data protection (micro-partitions and table types, storage metrics, Automatic Clustering, Time Travel and Fail-safe, zero-copy cloning for dev environments, and cross-region / cross-cloud replication), data governance (dynamic data masking, external tokenization, projection and aggregation policies, row access policies, tag-based masking, object tagging and classification, ACCESS_HISTORY lineage, data metric functions, the Horizon Catalog, and Data Clean Rooms), and data transformation (SQL ELT and MERGE, semi-structured VARIANT and FLATTEN, unstructured data and semantic views, Snowflake Cortex AI functions, UDFs / UDTFs / UDAFs and external functions, stored procedures, Snowpark DataFrames, and dev workflows with Notebooks, Git, and dbt Projects) — every question a real-world data engineering scenario with full explanations.

## Who Should Take This Exam?

The SnowPro Advanced: Data Engineer certification is designed for **data engineers, analytics engineers, and platform engineers** who build and operate production pipelines on the Snowflake AI Data Cloud. It validates advanced, hands-on skill across ingestion, pipeline design, performance tuning, storage protection, governance, and transformation. Candidates typically have 2+ years of Snowflake data-engineering experience.

**Prerequisites:** SnowPro Core Certified

**Typical study time:** 6-10 weeks of focused study (plus hands-on Snowflake pipeline experience)

## Exam Domains & Weights

The SnowPro Advanced: Data Engineer exam covers **5 domains**. Focus your study time based on the weights below — higher-weighted domains have more exam questions.

| Domain | Weight | Practice Qs |
|--------|--------|-------------|
| Data Movement | 28% | 70 |
| Data Transformation | 25% | 62 |
| Performance Optimization | 19% | 48 |
| Storage and Data Protection | 14% | 35 |
| Data Governance | 14% | 35 |
| **Total** | **100%** | **250** |

> 💡 **Study tip:** Data Movement (28%) is the heaviest domain — know exactly when to use Snowpipe vs Snowpipe Streaming vs the Kafka connector, how streams and tasks build continuous pipelines, and the difference between external, Iceberg, and hybrid tables. Data Transformation (25%) is about choosing the right pattern: dynamic tables vs streams-and-tasks, semi-structured FLATTEN, UDFs/UDTFs/UDAFs, Snowpark, and the newer Cortex AI functions. Performance Optimization (19%) rewards reading Query Profile (spilling, pruning) and choosing clustering vs Search Optimization vs Query Acceleration. Storage and Data Protection (14%) is Time Travel, Fail-safe, and zero-copy cloning. Data Governance (14%) covers masking, row access, projection and aggregation policies, classification, and Data Clean Rooms.
