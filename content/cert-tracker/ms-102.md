## Who is this exam for?

The MS-102 is the certification for Microsoft 365 administrators — the people who function as the integrating hub for all M365 workloads. You deploy and manage the tenant, coordinate across Teams, Exchange, SharePoint, security, and compliance, and work closely with identity and security teams.

This is a broad exam covering four major areas: tenant management, identity (Entra ID), security (Defender XDR), and compliance (Purview). The security domain is the largest at 30–35% — expect heavy coverage of Defender for Office 365, Defender for Endpoint, and Defender for Cloud Apps.

This exam was updated on **April 28, 2026** and now includes Microsoft 365 Backup and enhanced Defender XDR coverage. It also counts toward the **Microsoft 365 Certified: Administrator Expert** certification (paired with SC-300, MD-102, or MS-700).

### Deploy and manage a Microsoft 365 tenant (25–30%)

This domain covers the foundations of running a Microsoft 365 environment — creating tenants, managing domains, monitoring service health, managing users and groups, handling licensing, and configuring roles and delegated administration.

#### Implement and manage a Microsoft 365 tenant

- [Create a tenant](https://learn.microsoft.com/en-us/microsoft-365/admin/setup/setup)
- [Implement and manage domains](https://learn.microsoft.com/en-us/training/modules/manage-your-microsoft-365-tenant/3-manage-domains)
- [Configure org settings, including Security & privacy and Organization profile](https://learn.microsoft.com/en-us/microsoft-365/admin/manage/change-contact-preferences)
- [Monitor the health of Microsoft 365 services by using Service Health](https://learn.microsoft.com/en-us/training/modules/manage-your-microsoft-365-tenant/4-monitor-service-health)
- [Configure and review Network connectivity insights](https://learn.microsoft.com/en-us/microsoft-365/enterprise/microsoft-365-networking-overview)
- [Configure and monitor software updates by using the Microsoft 365 admin center](https://learn.microsoft.com/en-us/microsoft-365/admin/manage/release-options-in-office-365)
- [Monitor Microsoft 365 adoption and usage](https://learn.microsoft.com/en-us/microsoft-365/admin/activity-reports/activity-reports)
- [Configure and manage Microsoft 365 Backup](https://learn.microsoft.com/en-us/microsoft-365/backup/)

#### Manage users and groups

Day-to-day administration: creating users (including external/guest users), managing contacts, creating groups (including Microsoft 365 Groups and shared mailboxes), handling licence assignments with group-based licensing, and performing bulk operations via PowerShell.

- [Create and manage users in Microsoft Entra ID, including external users](https://learn.microsoft.com/en-us/training/modules/manage-user-accounts-licenses-microsoft-365/2-create-user-accounts)
- [Create and manage contacts in the Microsoft 365 admin center](https://learn.microsoft.com/en-us/microsoft-365/admin/misc/contacts)
- [Create and manage groups, including Microsoft 365 Groups and manage shared mailboxes](https://learn.microsoft.com/en-us/training/modules/manage-user-accounts-licenses-microsoft-365/4-manage-groups)
- [Manage and monitor Microsoft 365 licenses, including group-based licensing](https://learn.microsoft.com/en-us/training/modules/manage-user-accounts-licenses-microsoft-365/3-manage-user-account-licenses)
- [Perform bulk user management, including Microsoft Graph PowerShell and Microsoft Entra PowerShell](https://learn.microsoft.com/en-us/entra/identity/users/users-bulk-add)

#### Manage roles and role groups

RBAC in Microsoft 365 spans the M365 admin center, Entra ID, Defender, and Purview. You need to know built-in roles, how to delegate using administrative units, and how to use PIM for just-in-time admin access.

- [Implement and manage roles in Microsoft 365 and Microsoft Entra ID](https://learn.microsoft.com/en-us/training/modules/manage-your-microsoft-365-tenant/5-manage-admin-roles)
- [Manage permissions for Microsoft Defender XDR, Microsoft Purview and other workloads](https://learn.microsoft.com/en-us/defender-xdr/m365d-permissions)

- [Manage Microsoft Entra ID roles in PIM](https://learn.microsoft.com/en-us/entra/id-governance/privileged-identity-management/pim-configure)

### Manage security and threats by using Microsoft Defender XDR (30–35%)

This is the largest domain — a third of the exam. It covers the full Defender XDR suite: Secure Score, incident management, Defender for Office 365 (email protection), Defender for Endpoint (device security), and Defender for Cloud Apps (SaaS visibility). You need to know how to configure policies, investigate threats, and respond to incidents.

#### Review and respond to security reports and alerts generated by Microsoft Defender XDR

- [Review and respond to threats by using Microsoft Security Exposure Management, including Secure Score](https://learn.microsoft.com/en-us/defender-xdr/microsoft-secure-score)
- [Review and respond to incidents and alerts, including advanced hunting](https://learn.microsoft.com/en-us/defender-xdr/incidents-overview)
- [Review and respond to issues identified in Microsoft Defender XDR reports](https://learn.microsoft.com/en-us/defender-xdr/defender-xdr-reports)
- [Review and respond to threats identified by Microsoft Defender Threat Intelligence](https://learn.microsoft.com/en-us/defender/threat-intelligence/what-is-microsoft-defender-threat-intelligence-defender-ti)

#### Implement and manage email and collaboration protection by using Microsoft Defender for Office 365

Defender for Office 365 protects email and Teams against phishing, malware, and business email compromise. You need to know how to configure threat policies (Safe Links, Safe Attachments, anti-phishing), manage alerts, investigate threats, and run attack simulations.

- [Implement threat policies and rules in Defender for Office 365](https://learn.microsoft.com/en-us/defender-office-365/preset-security-policies)
- [Configure alert policies in Defender for Office 365](https://learn.microsoft.com/en-us/purview/alert-policies)
- [Investigate and respond to email and collaboration threats](https://learn.microsoft.com/en-us/defender-office-365/threat-explorer-real-time-detections-about)
- [Manage attack simulations, including training campaigns](https://learn.microsoft.com/en-us/defender-office-365/attack-simulation-training-get-started)
- [Manage restricted entities, including blocked users](https://learn.microsoft.com/en-us/defender-office-365/responding-to-a-compromised-email-account)

#### Implement and manage endpoint protection by using Microsoft Defender for Endpoint

- [Onboard devices to Microsoft Defender for Endpoint](https://learn.microsoft.com/en-us/defender-endpoint/onboarding)
- [Configure endpoint settings](https://learn.microsoft.com/en-us/defender-endpoint/configure-endpoints)
- [Review and respond to vulnerabilities in the Vulnerability Management dashboard](https://learn.microsoft.com/en-us/defender-vulnerability-management/tvm-dashboard-insights)

#### Implement and manage Microsoft Defender for Cloud Apps

- [Configure the app connector for Microsoft 365](https://learn.microsoft.com/en-us/defender-cloud-apps/connect-office-365)
- [Configure Defender for Cloud Apps policies, including triggering alerts](https://learn.microsoft.com/en-us/defender-cloud-apps/control-cloud-apps-with-policies)

- [Review and respond to issues identified in Cloud App Discovery](https://learn.microsoft.com/en-us/defender-cloud-apps/discovered-apps)

### Manage compliance by using Microsoft Purview (10–15%)

The smallest domain, but don't skip it — DLP and information protection questions are very practical. You need to know how to create and manage sensitivity labels, configure retention policies, set up DLP policies for M365 workloads, and respond to DLP alerts.

- [Implement sensitivity labels and sensitivity label policies](https://learn.microsoft.com/en-us/purview/sensitivity-labels)
- [Monitor label usage by using Content explorer, Activity explorer, and label reports](https://learn.microsoft.com/en-us/purview/data-classification-content-explorer)

#### Implement Microsoft Purview data loss prevention (DLP)

- [Configure DLP policies for M365 workloads, including Exchange, SharePoint, OneDrive, Teams, Power BI, and Copilot](https://learn.microsoft.com/en-us/purview/dlp-learn-about-dlp)
