## About the OCI-DATA-SCIENCE-PROFESSIONAL Exam

> Master the Oracle Cloud Infrastructure (OCI) 2026 Data Science Professional exam (1Z0-1110-26) — building, training, deploying and operating machine-learning models on the OCI Data Science service with the Accelerated Data Science (ADS) SDK, Oracle AutoML, the Model Catalog and Model Deployments, Jobs and ML Pipelines, OCI Generative AI and LangChain, and end-to-end MLOps.

The complete practice exam for the Oracle Cloud Infrastructure 2026 Data Science Professional certification (exam 1Z0-1110-26, the current 2026 version that replaces the retired 1Z0-1110-25). Covers OCI Data Science introduction and configuration (the managed-platform building blocks — Projects, Notebook Sessions, Models and the Model Catalog, Model Deployments, Jobs, ML Pipelines and conda environments; the Accelerated Data Science / ADS SDK; tenancy setup with IAM groups, dynamic groups, policies, resource principals and networking), designing and setting up the data science workspace (notebook session compute shapes and lifecycle, the managed JupyterLab, conda environments and the odsc conda CLI, storing secrets in OCI Vault, and Git Code Repositories), the end-to-end machine-learning lifecycle (fetching data from Object Storage and Autonomous Database, exploring and preparing data with ADS, visualization and profiling, training with scikit-learn, XGBoost, TensorFlow and PyTorch, Oracle AutoML, evaluating with the ADSEvaluator and explaining with the ADSExplainer and Oracle MLX, the Model Catalog and model artifacts with score.py and runtime.yaml, Model Deployments and the predict endpoint, OCI Generative AI and LangChain integration, and AI Quick Actions and Operators), applying MLOps practices (Jobs, scaling and autoscaling, model and deployment monitoring and logging, and ML Pipelines), and using related OCI services (OCI Data Flow managed Spark, OCI Open Data Service, and OCI Data Labeling) — every question a real-world data science scenario with full explanations.

## Who Should Take This Exam?

The OCI 2026 Data Science Professional (1Z0-1110-26) is designed for **data scientists, machine-learning engineers, and MLOps engineers** who build, train, deploy, and operate machine-learning and generative-AI models on Oracle Cloud Infrastructure — using the OCI Data Science service, the Accelerated Data Science (ADS) SDK, Oracle AutoML, the Model Catalog and Model Deployments, Jobs and ML Pipelines, and OCI Generative AI. It validates hands-on, end-to-end ML lifecycle skills, so **practical experience with Python, machine learning, and the OCI Data Science service (notebook sessions, the ADS SDK, model deployments) is strongly recommended**. It builds naturally on the OCI AI Foundations and OCI Foundations certifications.

**Prerequisites:** None formal (Oracle recommends Python and machine-learning fundamentals plus hands-on experience with the OCI Data Science service — notebook sessions, the ADS SDK, Model Catalog, Model Deployments, Jobs and Pipelines)

**Typical study time:** 8-12 weeks of intensive study

## Exam Domains & Weights

The OCI-DATA-SCIENCE-PROFESSIONAL exam covers **5 domains**. Focus your study time based on the weights below — higher-weighted domains have more exam questions.

| Domain | Weight | Practice Qs |
|--------|--------|-------------|
| OCI Data Science - Introduction and Configuration | 10% | 25 |
| Design and Set up Data Science Workspace | 15% | 38 |
| Implement end-to-end Machine Learning Lifecycle | 45% | 112 |
| Apply MLOps Practices | 20% | 50 |
| Use related OCI Services | 10% | 25 |
| **Total** | **100%** | **250** |

> 💡 **Study tip:** **Implement end-to-end Machine Learning Lifecycle** carries the most weight (45%) — nearly half the exam — so start there and go deep on the ADS SDK, training, AutoML, evaluation/explanation, the Model Catalog, Model Deployments, and OCI Generative AI. **Introduction and Configuration** and **Use related OCI Services** are the lightest (10% each), but don't skip them — exam questions can come from any domain.
