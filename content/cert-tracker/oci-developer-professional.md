The complete practice exam for the Oracle Cloud Infrastructure 2026 Developer Professional certification (exam 1Z0-1084-26, current 2026 version). Covers Cloud Native Fundamentals (cloud-native pillars, 12-factor apps, microservices architecture and design methodology), Cloud Native Applications and Containerization (Docker images and Dockerfiles, OCI Container Registry pull/push, image tags signing and scanning, container orchestration, building and deploying on Oracle Kubernetes Engine with Pods, Deployments, Services, Ingress, ConfigMaps and Secrets, Helm, autoscaling, VCN-native pod networking, persistent volumes, RBAC and workload identity), Serverless Technologies (Oracle Functions and the Fn FDK, API Gateway routes and request/response policies and authorizers, OCI Streaming with Apache Kafka compatibility, OCI Events service rules, OCI Queue asynchronous messaging), Testing and Securing Cloud Native Applications (cloud-native testing strategies, OCI Vault keys and secrets, envelope encryption, least-privilege IAM, dynamic groups and resource principals, image scanning and signing), and Monitoring and Troubleshooting (OCI Monitoring metrics and alarms, Logging and Logging Search, Application Performance Monitoring and distributed tracing) — every question a real-world cloud-native development scenario with full explanations.

## Who Should Take This Exam?

The OCI 2026 Developer Professional (1Z0-1084-26) is designed for **cloud developers, application developers, and DevOps engineers** who build, deploy, and operate cloud-native applications on Oracle Cloud Infrastructure — using containers and Oracle Kubernetes Engine (OKE), serverless Oracle Functions, API Gateway, Streaming, Events and Queue, and OCI Vault. It validates hands-on developer skills, so **practical experience with microservices, Docker, Kubernetes, and a programming language is strongly recommended**. It builds naturally on the OCI Foundations and Architect Associate certifications.

**Prerequisites:** None (OCI fundamentals plus hands-on experience with microservices, Docker, Kubernetes/OKE, Oracle Functions, and basic programming recommended)

**Typical study time:** 8-12 weeks of intensive study
