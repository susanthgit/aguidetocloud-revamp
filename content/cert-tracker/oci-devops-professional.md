## About the OCI-DEVOPS-PROFESSIONAL Exam

> Master the Oracle Cloud Infrastructure (OCI) 2026 DevOps Professional exam (1Z0-1109-26) — DevOps principles and microservices, containerization with Docker, OCIR and OCI Container Instances, infrastructure as code with Terraform and OCI Resource Manager, the OCI DevOps service with Code Repositories, Build and Deployment Pipelines, Artifacts and deployment strategies, Oracle Kubernetes Engine (OKE), DevSecOps with OCI Vault and image signing, and monitoring and observability with OCI Monitoring, Logging and Events.

The complete practice exam for the Oracle Cloud Infrastructure 2026 DevOps Professional certification (exam 1Z0-1109-26, current 2026 version). Covers Understanding DevOps Principles and Containerization Services (DevOps practices and microservices architecture, Docker images and Dockerfiles, OCI Container Registry pull/push, image signing and scanning, OCI Container Instances), Using Code and Templates for Provisioning Infrastructure (Infrastructure as Code with Terraform on OCI, OCI Resource Manager stacks and jobs, managed state and drift detection), Configuring and Managing CI/CD (the OCI DevOps service, Code Repositories and external Git mirroring, DevOps Artifacts and the OCI Artifact Registry, build_spec.yaml Build Pipelines, Deployment Pipelines, and rolling, blue-green and canary deployment strategies), Managing Containers with Oracle Kubernetes Engine (OKE clusters, managed and virtual node pools, kubeconfig and kubectl access, Pods, Deployments, Services, Ingress, VCN-native pod networking, persistent storage, autoscaling, cluster upgrades, and OKE security with workload identity and RBAC), Enabling DevSecOps (shift-left security, least-privilege IAM with dynamic groups and resource principals, OCI Vault keys and secrets, securing pipelines, and container image signing and verification), and Implementing Monitoring and Observability (DORA metrics, OCI Monitoring and MQL alarms, OCI Logging and Logging Search, OCI Events, Notifications and Application Performance Monitoring) — every question a real-world OCI DevOps scenario with full explanations.

## Who Should Take This Exam?

The OCI 2026 DevOps Professional (1Z0-1109-26) is designed for **DevOps engineers, platform engineers, and cloud developers** who automate build, test, and release on Oracle Cloud Infrastructure — using the OCI DevOps service for CI/CD, Oracle Kubernetes Engine (OKE), containers and OCIR, Terraform and OCI Resource Manager, OCI Vault, and OCI observability. It validates hands-on pipeline and platform skills, so **practical experience with Docker, Kubernetes, CI/CD pipelines, and Terraform is strongly recommended**. It builds naturally on the OCI Foundations and Architect Associate certifications.

**Prerequisites:** None (OCI fundamentals plus hands-on experience with Docker, Kubernetes/OKE, the OCI DevOps service, Terraform/Resource Manager, and OCI security recommended)

**Typical study time:** 8-12 weeks of intensive study
