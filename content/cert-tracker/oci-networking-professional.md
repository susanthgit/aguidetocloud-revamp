## About the OCI-NETWORKING-PROFESSIONAL Exam

> Master the Oracle Cloud Infrastructure (OCI) 2026 Networking Professional exam (1Z0-1124-26) — Virtual Cloud Network design and IPv4/IPv6 addressing, gateways and route tables, OCI Load Balancer and Network Load Balancer, public and private DNS with Traffic Management steering, hybrid connectivity with Dynamic Routing Gateways, FastConnect and Site-to-Site VPN, transitive routing, network security with Network Firewall, WAF and Zero Trust Packet Routing, workload migration connectivity, and network troubleshooting.

The complete practice exam for the Oracle Cloud Infrastructure 2026 Networking Professional certification (exam 1Z0-1124-26, current 2026 version). It covers Designing and Deploying OCI Virtual Cloud Networks (VCNs and subnets, IPv4/IPv6 addressing, the internet, NAT, service, local peering and dynamic routing gateways, route tables, security lists and Network Security Groups, and Service Gateway private access), Planning and Designing OCI Networking Solutions and App Services (IP address management, the Flexible Load Balancer and Network Load Balancer, public and private DNS, and Traffic Management steering policies with health checks), Designing Hybrid Networking Architectures (Dynamic Routing Gateway attachments and route distributions, BGP, Site-to-Site IPSec VPN, FastConnect provider, third-party and colocation models, multicloud interconnect, and IPSec over FastConnect), Transitive Routing (DRG hub-and-spoke transit, legacy local-peering-gateway transit, and routing through a network virtual appliance), Implementing and Operating Secure OCI Networking (inter-tenancy and cross-compartment connectivity with Define/Endorse/Admit IAM policies, the Bastion service and Cloud Shell, OCI Network Firewall, Web Application Firewall and the Certificates service, and Zero Trust Packet Routing), Migrating Workloads to OCI (optimal connectivity strategies from on-premises and from another cloud provider, and multicloud scenarios), and Troubleshooting OCI Networking and Connectivity Issues (Network Path Analyzer, VCN Flow Logs, VTAP, Network Visualizer, and connectivity and hybrid-tunnel diagnostics) — every question a real-world OCI networking scenario with full explanations.

## Who Should Take This Exam?

The OCI 2026 Networking Professional (1Z0-1124-26) is designed for **cloud network engineers, network architects, and hybrid-connectivity specialists** who plan, design, implement, operate, migrate, and troubleshoot networking on Oracle Cloud Infrastructure — across VCN design, load balancing and DNS, hybrid connectivity with DRG/FastConnect/VPN, transitive routing, network security, and workload migration. It validates hands-on networking skills, so **practical experience with OCI VCNs, gateways, routing, FastConnect/VPN, load balancing, and the OCI network-security services (Network Firewall, WAF, Zero Trust Packet Routing) is strongly recommended**. It builds naturally on the OCI Foundations and Architect Associate certifications.

**Prerequisites:** None (OCI fundamentals plus hands-on experience with OCI networking, VCNs, and hybrid connectivity recommended)

**Typical study time:** 6-10 weeks of focused study

Together they are 60% of the exam, so start there. The lighter 10% domains — VCN design, Transitive Routing, Migration, and Troubleshooting — round out the rest, but VCN design frames everything else, so don't skip it.
