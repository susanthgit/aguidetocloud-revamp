## About the OCI Enterprise AI Professional Exam

> Build, deploy, secure, and operate enterprise AI solutions on OCI - from model selection and customization to Responses API agents, hosted applications, guardrails, and cost control.

This practice bank contains 250 original questions for the **Oracle Cloud Infrastructure Enterprise AI Professional**
certification, exam **1Z0-1158-26**. The exam tests the full enterprise-AI lifecycle on OCI: applying LLM and agentic
concepts, configuring offered or imported models, building agents with OCI's OpenAI-compatible Responses API and
managed tools, deploying custom agent code as hosted applications, and governing the result with IAM, private
networking, guardrails, observability, and cost controls.

## Who Should Take This Exam?

The certification is aimed at **AI engineers, agent developers, machine-learning engineers, platform engineers,
cloud architects, security engineers, and technical leads** who build production AI applications on OCI.

You should be comfortable choosing between prompting, retrieval, imported models, and fine-tuning; working with
agent tools and memory; and reasoning about production deployment, security, reliability, and cost.

**Prerequisites:** None required. Hands-on OCI Generative AI and agent-development experience is strongly recommended.

**Typical study time:** 5-8 weeks of focused study, depending on your OCI and agentic-AI experience.

## What to Study First

1. **Separate the model choices** - on-demand versus dedicated, offered versus imported, prompting versus retrieval versus fine-tuning, and fine-tuning versus hosting clusters.
2. **Learn the agent API layers** - Responses generates work, Conversations carries state, Projects isolate resources and memory, and Files/Vector Stores ground retrieval.
3. **Map each tool to its job** - File Search, Code Interpreter, Function Calling, and MCP Calling solve different agent requirements.
4. **Understand hosted applications** - application settings, container-image deployments, active versions, replicas, autoscaling, managed storage, networking, authentication, and endpoints.
5. **Treat governance as an architecture decision** - least privilege, private access, retention, guardrails, observability, and cost controls belong in the design from the start.
6. **Keep the database exam separate** - VECTOR, DBMS_VECTOR_CHAIN, Select AI, and the Autonomous Database MCP Server belong to 1Z0-1159-26, not this exam.

## Oracle AI Certification Path

A practical sequence is **OCI AI Foundations Associate** for the broad concepts, then **OCI Enterprise AI Professional**
for the cloud platform and agent lifecycle. Add **Agentic AI for Oracle AI Database Professional** if your work moves
agent reasoning, retrieval, and tool access into Oracle AI Database.
