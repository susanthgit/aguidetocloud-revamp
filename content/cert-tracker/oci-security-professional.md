## Who Should Take This Exam?

The OCI 2026 Security Professional (1Z0-1104-26) is designed for **cloud security engineers, security architects, and platform/SecOps teams** who design, implement, and operate security controls on Oracle Cloud Infrastructure — across identity and access, network and infrastructure security, application and data protection, cloud security posture, and security operations. It validates hands-on security skills, so **practical experience with OCI IAM, networking, and the OCI security services (Cloud Guard, Vault, WAF, Bastion, Vulnerability Scanning) is strongly recommended**. It builds naturally on the OCI Foundations and Architect Associate certifications.

**Prerequisites:** None (OCI fundamentals plus hands-on experience with OCI IAM, networking, and security services recommended)

**Typical study time:** 6-10 weeks of focused study
