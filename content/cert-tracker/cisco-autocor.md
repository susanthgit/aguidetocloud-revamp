## About the AUTOCOR Exam

> Master network automation on Cisco platforms — build it, ship it with CI/CD, operate it, and augment it with AI

The 350-901 AUTOCOR (Designing, Deploying, and Managing Network Automation Systems) is the **core exam for the new Cisco Certified Network Professional (CCNP) Automation and CCIE Automation certifications** — Cisco's 2026 successor to the DevNet Professional track. It validates your ability to design, build, deploy, operate, and secure network automation systems: writing automation with Ansible, Terraform, RESTCONF over a YANG model, and Python; consuming REST APIs with pagination, OAuth2, and rate limiting; managing infrastructure as code with Git and GitLab CE CI/CD, Cisco Modeling Labs (CML), Docker Compose, and a source of truth; operating it with model-driven telemetry, logging, pyATS change validation, CA-signed TLS, and secure coding; and applying **AI in automation** — building an MCP server with Python FastMCP and an LLM-driven conversational agent, and evaluating AI accuracy and risk. Passing AUTOCOR earns the Cisco Certified Automation Specialist - Core badge. Original practice questions. Not affiliated with, endorsed by, or sourced from Cisco Systems certification exams.

## Who Should Take This Exam?

AUTOCOR is designed for **network automation engineers, NetDevOps and platform engineers, and SREs who design and operate automation across Cisco infrastructure**. It suits candidates pursuing CCNP Automation or CCIE Automation. Cisco recommends 3-5 years of networking experience plus hands-on time with Python, Ansible/Terraform, Git/CI-CD, and Cisco programmability (NETCONF/RESTCONF/YANG, NSO, pyATS).

**Typical study time:** 8-12 weeks of focused study

## Exam Domains & Weights

The AUTOCOR exam covers **4 domains**. Focus your study time based on the weights below — higher-weighted domains have more exam questions.

| Domain | Weight | Practice Qs |
|--------|--------|-------------|
| Network Automation | 30% | 75 |
| Infrastructure as Code | 30% | 75 |
| Operations | 20% | 50 |
| AI in Automation | 20% | 50 |
| **Total** | **100%** | **250** |

> 💡 **Study tip:** **Network Automation** and **Infrastructure as Code** each carry 30% — together that is 60% of the exam, so start there. Be able to choose the right tool for a task (Ansible's procedural model vs Terraform's declarative state vs raw RESTCONF/Python) and to read a `.gitlab-ci.yml`, a Docker Compose file, and a YANG-to-JSON payload cold. The new **AI in Automation** domain (20%) is fresh in v2.0 — know how an MCP server (FastMCP) exposes tools to an agent, and how to keep AI output validated and human-in-the-loop before it touches the network.

## Cisco Certification Path

Cisco's new Automation track follows: Associate → Professional (CCNP Automation) → Expert (CCIE Automation). **AUTOCOR 350-901** is the core exam; CCNP Automation requires AUTOCOR plus **one concentration** — 300-435 ENAUTO (Enterprise) or 300-635 DCNAUTO (Data Center). AUTOCOR replaces the retired DevNet Professional core (DEVCOR) under Cisco's 2026 certification refresh.
