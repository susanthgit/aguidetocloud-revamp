faq:
  - question: "What is the Microsoft 365 Copilot Frontier programme?"
    answer: "The Frontier programme is Microsoft's opt-in early-access programme that lets organisations trial experimental and upcoming Copilot AI capabilities before general availability. Features in Frontier are production-quality but may evolve, change, or be withdrawn based on real-world feedback. Tenant admins enable it via the Microsoft 365 Admin Centre."
  - question: "How do I enrol my organisation in the Frontier programme?"
    answer: "Go to the Microsoft 365 Admin Centre → Settings → Org settings → Copilot → Frontier programme. An admin must explicitly opt in. Some features (like Copilot Cowork) also require approving Anthropic as a sub-processor."
  - question: "What is the three-tier release model (Frontier, Standard, Deferred)?"
    answer: "In 2026, Microsoft introduced three release tiers for Copilot features. Frontier gives early access to experimental features. Standard is the default — features arrive at GA automatically. Deferred lets organisations delay new features by ~30 days after GA for internal testing and change management."
  - question: "Does enrolling in Frontier guarantee early access to everything?"
    answer: "No. Some Frontier features have additional requirements — for example, Copilot Cowork requires approving Anthropic as a sub-processor, and Call Delegation requires a Teams Phone licence. Each feature may also roll out in stages across tenants."
  - question: "Will Frontier features definitely become generally available?"
    answer: "Not necessarily. Microsoft explicitly states that Frontier features can change, pause, or be withdrawn. GA timing is never guaranteed. However, most Frontier features do eventually graduate to GA — examples include Scheduled Prompting and Copilot Notebooks enhancements."
  - question: "What about data residency and Anthropic for EU tenants?"
    answer: "Features using Anthropic Claude (like Copilot Cowork and multi-model Researcher) require data to cross to Anthropic's infrastructure, which is excluded from the EU Data Boundary. EU/EFTA/UK tenants must explicitly opt in to Anthropic as a sub-processor — it's disabled by default in these regions."

    answer: "We verify all features against public Microsoft sources regularly. Each feature card shows a 'last verified' date. When Microsoft announces changes — new Frontier features, GA graduations, or withdrawals — we update within days."
