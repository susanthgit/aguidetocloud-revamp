## When per-seat is enough vs when consumption kicks in

A decision framework based on typical Cowork use patterns.

| Pattern | Likely model |
|---|---|
| Light Cowork use (a few tasks per week per user) | Per-seat covers it |
| Heavy Cowork use (multiple long-running tasks daily) | Per-seat + consumption |
| Mixed team (some heavy users, mostly light) | Per-seat + consumption with admin caps |
