## What You'll Learn

This full AI-900 course prepares you for the **Azure AI Fundamentals** certification. You'll understand:

- **Core AI concepts** — what AI, machine learning, and deep learning actually are
- **Machine learning fundamentals** — regression, classification, clustering, and how models learn from data
- **Computer vision** — image analysis, object detection, OCR, and Azure AI Vision
- **Natural Language Processing (NLP)** — text analysis, sentiment detection, language understanding, and Azure AI Language
- **Generative AI** — Azure OpenAI Service, GPT models, DALL-E, and responsible AI practices
- **Responsible AI principles** — fairness, reliability, safety, privacy, inclusiveness, transparency, and accountability

> **Tip:** No coding or AI experience required. This is a fundamentals exam — it tests concepts and awareness, not implementation.

## Key Topics Covered

### AI Workloads and Considerations (15-20%)

- **What is AI?** — Systems that mimic human intelligence — learning, reasoning, and self-correction
- **Machine learning vs rule-based systems** — ML learns from data; rules are hard-coded by humans
- **Responsible AI** — Microsoft's six principles: fairness, reliability & safety, privacy & security, inclusiveness, transparency, accountability
- **Common AI workloads** — Anomaly detection, computer vision, NLP, conversational AI, generative AI

### Machine Learning on Azure (20-25%)

- **Supervised learning** — The model learns from labelled data (regression predicts numbers, classification predicts categories)
- **Unsupervised learning** — The model finds patterns in unlabelled data (clustering groups similar items)
- **Azure Machine Learning** — Microsoft's platform for building, training, and deploying ML models
- **Automated ML (AutoML)** — Azure automatically selects the best algorithm and hyperparameters for your data
- **Model evaluation** — Accuracy, precision, recall, F1 score, and confusion matrices

### Computer Vision on Azure (15-20%)

- **Azure AI Vision** — Analyse images for content, objects, faces, and text
- **Custom Vision** — Train your own image classification or object detection models without writing code
- **OCR (Optical Character Recognition)** — Extract text from images and scanned documents
- **Azure AI Video Indexer** — Extract insights from video content (faces, topics, sentiment, transcription)

### Natural Language Processing (15-20%)

- **Azure AI Language** — Text analytics, sentiment analysis, key phrase extraction, named entity recognition
- **Azure AI Translator** — Real-time translation across 100+ languages
- **Azure AI Speech** — Speech-to-text, text-to-speech, and speech translation
- **Conversational AI** — Build chatbots with Azure Bot Service and Copilot Studio
- **Language Understanding (CLU)** — Train models to understand natural language intents and entities

### Generative AI on Azure (15-20%)

- **Azure OpenAI Service** — Access GPT-4, GPT-4o, DALL-E, and Whisper models through Azure
- **Prompts and completions** — How large language models generate text from prompts
- **Responsible generative AI** — Content filtering, grounding, and guardrails
- **RAG (Retrieval-Augmented Generation)** — Combining LLMs with your own data for accurate, grounded responses
- **Copilots** — AI assistants built on top of foundation models (Microsoft 365 Copilot, GitHub Copilot)

## Study Tips for the AI-900

1. **Don't overthink the maths** — the exam tests concepts, not formulas
2. **Know the Azure services by name** — understand what each service does (Vision, Language, Speech, OpenAI)
3. **Understand responsible AI deeply** — this topic appears throughout the exam
4. **Try the Azure AI services** — many have free tiers you can experiment with
5. **Focus on generative AI** — this topic has grown significantly in recent exam updates

## Frequently Asked Questions

**Is AI-900 easy?**
Yes — it's one of the easiest Microsoft certifications. It covers concepts and awareness, not hands-on implementation. Most people with basic tech understanding can pass with 1-2 weeks of study.

**Do I need programming skills for AI-900?**
No — this is a conceptual exam. You don't need to write any code.

**Is AI-900 worth it in 2026?**
Absolutely — with AI transforming every industry, having a validated understanding of AI concepts is valuable for any role. It's also a great foundation before pursuing AI-102 (Azure AI Engineer) or the Microsoft 365 Copilot certifications.

**What's the difference between AI-900 and AI-102?**
AI-900 covers AI **concepts** (fundamentals). AI-102 covers **building** AI solutions on Azure (associate level, requires coding).

**Can I get a free AI-900 exam voucher?**
Yes — attend a [Microsoft Virtual Training Day for AI Fundamentals](https://events.microsoft.com/en-us/mvtd) and you'll receive a free exam voucher.
