## What You'll Learn

This full DP-900 course covers the **Azure Data Fundamentals** certification. You'll understand:

- **Core data concepts** — Relational vs non-relational data, data warehousing, batch vs streaming, and data roles
- **Relational data on Azure** — Azure SQL Database, Azure SQL Managed Instance, and Azure Database for MySQL/PostgreSQL
- **Non-relational data on Azure** — Azure Cosmos DB, Azure Table Storage, Azure Blob Storage, and Azure Files
- **Analytics on Azure** — Azure Synapse Analytics, Azure Databricks, Azure Data Factory, Power BI, and real-time analytics

## Key Topics Covered

### Core Data Concepts (25-30%)

- **Data formats** — Structured (tables), semi-structured (JSON, XML), and unstructured (images, video)
- **Data workloads** — OLTP (transactions), OLAP (analytics), batch processing, and streaming
- **Data roles** — Database Administrator, Data Engineer, Data Analyst, and Data Scientist
- **Data storage** — File systems, databases, data lakes, and data warehouses

### Relational Data on Azure (20-25%)

- **Azure SQL Database** — Fully managed SQL Server in the cloud with built-in HA and scaling
- **Azure SQL Managed Instance** — Near-100% SQL Server compatibility for lift-and-shift migrations
- **Azure Database for MySQL/PostgreSQL** — Managed open-source database engines
- **SQL fundamentals** — SELECT, INSERT, UPDATE, DELETE, JOINs, and aggregations

### Non-Relational Data on Azure (15-20%)

- **Azure Cosmos DB** — Multi-model, globally distributed database with multiple APIs (SQL, MongoDB, Cassandra, Table, Gremlin)
- **Azure Table Storage** — Key-value NoSQL storage for semi-structured data
- **Azure Blob Storage** — Object storage for documents, images, backups, and media

### Analytics on Azure (25-30%)

- **Azure Synapse Analytics** — Unified analytics platform combining data warehousing and big data analytics
- **Azure Data Factory** — Cloud ETL/ELT service for data integration pipelines
- **Azure Databricks** — Apache Spark-based analytics platform for data engineering and machine learning
- **Power BI** — Business intelligence tool for interactive dashboards and reports
- **Azure Stream Analytics** — Real-time event processing and analytics

## Frequently Asked Questions

**Is DP-900 easy?**
Yes — it's fundamentals-level and covers concepts, not hands-on implementation. 1-2 weeks of study is typical.

**Do I need SQL knowledge for DP-900?**
Basic SQL understanding helps but isn't strictly required. The exam covers SQL at a conceptual level.

**What comes after DP-900?**
DP-203 (Data Engineer), DP-300 (Database Administrator), or PL-300 (Power BI Data Analyst).

### Certification Renewal

Microsoft role-based and specialty certifications **expire after one year**. To renew:

1. You'll receive an email 6 months before expiry
2. Complete a free online renewal assessment on Microsoft Learn
3. No exam centre visit required — it's done online
4. You can attempt the renewal assessment as many times as needed

> Fundamentals certifications (AZ-900, AI-900, SC-900, etc.) **do not expire** — they're valid for life.
