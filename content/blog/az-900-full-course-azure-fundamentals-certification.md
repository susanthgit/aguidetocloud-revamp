## What You'll Learn

This complete AZ-900 course covers everything for the **Microsoft Azure Fundamentals** certification — the most popular starting point for anyone entering cloud computing. You'll understand:

- **Cloud computing concepts** — IaaS, PaaS, SaaS, public/private/hybrid cloud, and the shared responsibility model
- **Core Azure services** — Virtual Machines, App Service, Azure Functions, Storage, SQL Database, Cosmos DB, and Virtual Networks
- **Azure management tools** — Azure Portal, CLI, PowerShell, Cloud Shell, and Azure Arc
- **Security and identity** — Microsoft Entra ID, RBAC, NSGs, Azure Firewall, and Microsoft Defender for Cloud
- **Governance and compliance** — Azure Policy, Resource Locks, Blueprints, and Management Groups
- **Cost management** — Pricing calculator, TCO calculator, Cost Management, and reserved instances

> **Tip:** This is the #1 recommended starting certification for anyone new to Azure or cloud computing. Zero prior experience needed.

## Key Topics Covered

### Cloud Concepts (25-30%)

Understanding cloud computing is the foundation of everything in Azure:

- **Why cloud?** — Scalability, elasticity, high availability, disaster recovery, and reduced capital expenditure
- **Cloud models** — Public cloud (Azure, AWS, GCP), private cloud (on-premises), and hybrid cloud (combination)
- **Service models** — IaaS (you manage the OS and apps), PaaS (you manage the app), SaaS (everything managed for you)
- **Shared responsibility** — Microsoft manages physical security and infrastructure; you manage data, identities, and applications
- **Consumption-based pricing** — Pay only for what you use, unlike traditional on-premises where you buy hardware upfront

### Core Azure Services (20-25%)

- **Compute** — Virtual Machines (IaaS), App Service (PaaS), Azure Functions (serverless), Azure Kubernetes Service (containers)
- **Networking** — Virtual Networks, Subnets, VPN Gateway, ExpressRoute, Load Balancer, Application Gateway
- **Storage** — Blob Storage (objects), File Storage (file shares), Queue Storage (messages), Table Storage (NoSQL), Disk Storage (VMs)
- **Databases** — Azure SQL Database, Azure Cosmos DB, Azure Database for MySQL/PostgreSQL
- **Azure Marketplace** — Pre-built solutions and virtual appliances from Microsoft and third-party vendors

### Security, Identity, and Governance (30-35%)

- **Microsoft Entra ID** — Cloud identity service for authentication and single sign-on
- **Multi-factor authentication (MFA)** — Extra layer of security beyond passwords
- **Role-Based Access Control (RBAC)** — Assign specific permissions to users, groups, and service principals
- **Azure Policy** — Enforce organisational standards across your Azure environment
- **Resource Locks** — Prevent accidental deletion or modification of critical resources
- **Management Groups** — Organise subscriptions for policy and governance at scale

### Cost Management (20-25%)

- **Factors affecting cost** — Region, resource type, usage, and reserved capacity
- **Azure Pricing Calculator** — Estimate costs before deploying
- **TCO Calculator** — Compare on-premises costs with Azure
- **Azure Cost Management** — Monitor, allocate, and optimise your spending
- **Reserved Instances** — Save up to 72% by committing to 1 or 3-year terms

## Frequently Asked Questions

**Is AZ-900 easy?**
Yes — it's designed for complete beginners. With 1-2 weeks of focused study, most people pass on their first attempt.

**Do I need Azure experience for AZ-900?**
No — the exam covers concepts and awareness. No hands-on experience or coding is required.

**Can I get the AZ-900 exam for free?**
Yes — attend a [Microsoft Virtual Training Day](https://events.microsoft.com/en-us/mvtd) and receive a free exam voucher. Check for "Azure Fundamentals" sessions.

**What should I study after AZ-900?**
The most common paths are AZ-104 (Azure Administrator), AZ-204 (Azure Developer), or AI-900 (AI Fundamentals).

**How long does it take to study for AZ-900?**
1-2 weeks of 2-3 hours per day for most beginners. If you already have IT experience, you may need even less time.
