## What Is Copilot Notebook?

**Copilot Notebook** is a dedicated workspace within Microsoft 365 Copilot designed for **long-form, complex AI interactions** — the kind that don't fit neatly into a quick chat prompt.

Think of it this way:

| Feature | Copilot Chat | Copilot Notebook |
|---------|-------------|-----------------|
| **Prompt length** | Short prompts (a few sentences) | Long, detailed prompts (up to 18,000 characters) |
| **Best for** | Quick questions, summaries, drafts | Deep analysis, multi-step instructions, document processing |
| **Output** | Conversational replies | Structured, editable output with full formatting |
| **Context** | Chat-based (back and forth) | Persistent workspace — add references, files, and instructions |
| **Collaboration** | Private to you | **Shareable** — invite others to collaborate |

> In simple terms: **Copilot Chat is like texting. Copilot Notebook is like writing a detailed brief and getting a comprehensive response.**

---

## Where to Find Copilot Notebook

You can access Copilot Notebook from:

1. **Microsoft 365 Copilot app** — Click the **Notebooks** icon in the left navigation panel
2. **microsoft365.com/chat** — Switch to the Notebook tab at the top
3. **Microsoft 365 Copilot mobile app** — Available on iOS and Android

Copilot Notebook is available to users with a **Microsoft 365 Copilot licence** ($30/user/month).

## Key Features

### 1. Extended Prompt Length (18,000 Characters)

Standard Copilot Chat limits prompts to around 2,000 characters. Notebook supports up to **18,000 characters** — that's roughly 4-5 pages of text. This means you can paste entire documents, email threads, or meeting transcripts and ask Copilot to analyse them.

### 2. File References and Grounding

Use the **/** (slash) or **paperclip icon** to attach files from OneDrive and SharePoint directly in your prompt. Copilot reads the file content and uses it to ground its response.

**Example:** "Based on /Q1-Sales-Report.xlsx and /Marketing-Strategy.docx, create a summary comparing our sales performance against our marketing targets."

### 3. Persistent Workspace

Unlike chat (which is ephemeral), Notebook prompts and responses are **saved automatically**. You can return to any notebook later, refine your prompt, and regenerate the output — without starting over.

### 4. Shareable Notebooks

You can **share a Copilot Notebook** with colleagues. All references, instructions, and context in the shared notebook are accessible to collaborators, while your individual chat conversations remain private.

This is powerful for teams — create a shared notebook with standard prompts and references that anyone on the team can use.

### 5. Overview Page

Each Notebook automatically creates a **summary overview** that captures key information and updates as you add more content. It also suggests related notebooks and files to help you reuse work.

### 6. Podcast and Audio Generation

One of the most unique features — you can ask Copilot Notebook to **generate a podcast-style audio summary** of your content. Copilot creates an engaging narrated audio overview that you can listen to on the go.

**Example:** "Create an audio podcast summarising the key takeaways from this project report, in a conversational tone with two speakers."

### 7. PDF and Document Analysis

Upload or reference PDFs, Word documents, Excel files, and PowerPoint presentations. Copilot can:

- **Summarise** long documents in your preferred format
- **Extract** key data points, dates, and action items
- **Compare** multiple documents side by side
- **Translate** content across languages
- **Reformat** content (e.g., turn a report into bullet points or a table)

## Real-World Use Cases

| Scenario | How to Use Notebook |
|----------|-------------------|
| **Meeting prep** | Paste last month's meeting notes + agenda → "Create a briefing document with key decisions and open action items" |
| **Research synthesis** | Reference 3-4 source documents → "Synthesise these into a 2-page executive summary with recommendations" |
| **Policy review** | Paste a 20-page policy document → "Summarise the key changes from the previous version and flag any compliance risks" |
| **Training material** | Reference a technical document → "Create a beginner-friendly explanation with examples and a quiz" |
| **Content creation** | Provide an outline + reference docs → "Write a blog post following this structure, using data from the attached files" |
| **Podcast creation** | Reference a report → "Generate a 5-minute audio overview in a conversational podcast style" |

## How Notebook Differs from Copilot Pages

| Feature | Copilot Notebook | Copilot Pages |
|---------|-----------------|---------------|
| **Purpose** | Long-form prompting and AI generation | Collaborative document editing |
| **Input** | You write the prompt, Copilot generates output | Copilot generates a starting point, you edit collaboratively |
| **Collaboration** | Share the entire notebook (read/prompt access) | Real-time co-editing like a Loop component |
| **Best for** | Complex analysis, multi-file synthesis | Team brainstorming, shared drafts |

## Tips for Getting the Most from Notebook

1. **Be specific in your prompt** — the more detail you provide, the better the output. Notebook handles long prompts well, so don't hold back
2. **Reference multiple files** — Notebook shines when you give it multiple sources to synthesise
3. **Iterate on the output** — regenerate with refined instructions rather than starting a new notebook
4. **Use custom instructions** — tell Copilot your preferred format: "Always respond in British English, use bullet points, and include a summary table at the top"
5. **Share with your team** — create standard notebooks for recurring tasks (weekly reports, customer briefings, etc.)
6. **Try audio generation** — the podcast feature is surprisingly useful for catching up on long documents while commuting

## Frequently Asked Questions

**Do I need a paid licence for Copilot Notebook?**
Yes — Copilot Notebook requires a Microsoft 365 Copilot licence ($30/user/month). It's not available in the free Copilot Chat.

**How is Notebook different from just using a long prompt in Chat?**
Notebook supports much longer prompts (18,000 vs ~2,000 characters), saves your work automatically, allows file references, and can be shared with colleagues. Chat is ephemeral and conversational.

**Can I share a Notebook with external users?**
Currently, Notebook sharing works within your organisation (same Microsoft 365 tenant). External sharing is not supported.

**Does Notebook have access to my company's data?**
Yes — like all Microsoft 365 Copilot features, Notebook can access data you have permission to see in Microsoft Graph (emails, files, chats, meetings). It follows your existing security permissions.

**Can I generate audio/podcasts from any document?**
Yes — you can ask Notebook to create an audio overview from any referenced document. The audio feature supports different styles, tones, and speaker configurations.

**How many Notebooks can I create?**
There's no hard limit on the number of notebooks you can create. Each notebook is saved automatically and accessible from the Notebooks section in the left navigation.
