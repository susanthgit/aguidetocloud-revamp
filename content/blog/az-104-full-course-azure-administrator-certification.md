## What You'll Learn

This full AZ-104 course prepares you for the **Microsoft Azure Administrator Associate** — one of the most in-demand cloud certifications. You'll master:

- **Identity and governance** — Microsoft Entra ID, RBAC, Azure Policy, Management Groups, subscriptions
- **Storage** — Blob, File, Table, Queue storage; replication options; Azure Files and Azure File Sync
- **Compute** — Virtual Machines, VM scale sets, App Service, Azure Container Instances
- **Networking** — Virtual Networks, subnets, NSGs, VPN Gateway, ExpressRoute, Load Balancer, Application Gateway
- **Monitoring** — Azure Monitor, Log Analytics, alerts, Network Watcher, and resource health

> **Tip:** This is the most popular associate-level Azure cert. Having hands-on Azure experience (even via free account) will significantly help.

## Key Topics Covered

### Identity and Governance (20-25%)

- **Microsoft Entra ID** — Users, groups, administrative units, and dynamic membership
- **RBAC** — Built-in roles, custom roles, and scope hierarchy (Management Group → Subscription → Resource Group → Resource)
- **Azure Policy** — Enforce tagging, allowed regions, and resource type restrictions
- **Azure Blueprints** — Package policies, RBAC, and ARM templates for repeatable governance
- **Cost management** — Budgets, alerts, and cost analysis in Cost Management + Billing

### Storage (15-20%)

- **Storage accounts** — Performance tiers (Standard vs Premium), redundancy (LRS, GRS, ZRS, RA-GRS)
- **Blob storage** — Containers, access tiers (Hot/Cool/Archive), lifecycle management
- **Azure Files** — SMB and NFS file shares in the cloud; Azure File Sync for hybrid
- **Storage security** — Shared access signatures (SAS), stored access policies, private endpoints, and encryption

### Compute (20-25%)

- **Virtual Machines** — Sizes, availability sets, availability zones, proximity placement groups
- **VM Scale Sets** — Auto-scaling based on metrics or schedule
- **Azure App Service** — Web apps, deployment slots, scaling, and custom domains
- **Azure Container Instances** — Run containers without managing infrastructure
- **Azure Kubernetes Service (AKS)** — Managed Kubernetes for container orchestration

### Networking (25-30%)

- **Virtual Networks** — Address spaces, subnets, DNS settings, and service endpoints
- **Network Security Groups (NSGs)** — Inbound/outbound rules, priority, and application security groups
- **VPN Gateway** — Site-to-site, point-to-site, and VNet-to-VNet VPN connections
- **Azure Load Balancer** — Layer 4 load balancing for VMs
- **Application Gateway** — Layer 7 load balancing with WAF, SSL termination, and URL routing
- **Azure DNS** — Host DNS zones and manage records in Azure

### Monitoring (10-15%)

- **Azure Monitor** — Metrics, logs, dashboards, and workbooks
- **Log Analytics** — Collect and query logs with KQL (Kusto Query Language)
- **Alerts** — Action groups, alert rules, and smart detection
- **Network Watcher** — Connection troubleshooting, IP flow verify, and NSG diagnostics

## Frequently Asked Questions

**Is AZ-104 harder than AZ-900?**
Yes — AZ-104 is associate-level and expects hands-on Azure administration knowledge. AZ-900 is concepts only.

**Do I need to memorise PowerShell and CLI commands?**
You should understand the key commands and their purposes. The exam may show code snippets and ask what they do.

**What salary can I expect with AZ-104?**
Azure Administrators typically earn $80,000-$130,000+ depending on experience and location (source: various salary surveys).

**What should I study after AZ-104?**
Popular paths: AZ-500 (Security), AZ-400 (DevOps), or AZ-305 (Solutions Architect).
