## What You'll Learn

This full AZ-500 course prepares you for the **Microsoft Azure Security Engineer Associate** certification. You'll master:

- **Identity and access** — Entra ID, managed identities, service principals, RBAC, PIM, and Conditional Access
- **Network security** — NSGs, Azure Firewall, DDoS Protection, Private Link, and service endpoints
- **Compute security** — VM security, container security, and Azure Kubernetes Service hardening
- **Data and application security** — Key Vault, storage encryption, SQL security, and Azure App Configuration
- **Security operations** — Microsoft Defender for Cloud, Sentinel, log analytics, and security alerts

## Key Topics Covered

### Identity and Access (25-30%)

- **Microsoft Entra ID** — Tenant-level security, Conditional Access, PIM, and Identity Protection
- **Managed identities** — System-assigned vs user-assigned identities for Azure resources
- **Azure Key Vault** — Secrets, keys, and certificates management with RBAC and access policies
- **External identities** — B2B collaboration and Azure AD B2C for customer-facing applications

### Network Security (20-25%)

- **Network Security Groups** — Inbound/outbound rules, application security groups, and flow logs
- **Azure Firewall** — Centralised, stateful firewall with FQDN filtering and threat intelligence
- **Azure DDoS Protection** — Standard and Network tier for volumetric attack mitigation
- **Private Link and Private Endpoints** — Access Azure services over private IP addresses
- **Azure Bastion** — Secure RDP/SSH access to VMs without public IP addresses

### Compute Security (20-25%)

- **VM security** — Disk encryption, just-in-time VM access, endpoint protection, and update management
- **Container security** — Azure Container Registry scanning, AKS pod security, and image signing
- **Azure App Service security** — Authentication, TLS/SSL, and managed certificates

### Data Protection (15-20%)

- **Storage security** — Encryption at rest, customer-managed keys, and immutable storage
- **Database security** — Transparent data encryption (TDE), Always Encrypted, dynamic data masking, and auditing
- **Azure Information Protection** — Classification and labelling of sensitive data

### Security Operations (15-20%)

- **Microsoft Defender for Cloud** — Security posture management, recommendations, and Secure Score
- **Microsoft Sentinel** — SIEM and SOAR for threat detection and automated response
- **Azure Monitor security** — Log Analytics queries, diagnostic settings, and alert rules

## Frequently Asked Questions

**Is AZ-500 harder than AZ-104?**
It's a similar difficulty level but focuses specifically on security. If you have AZ-104 knowledge, you're well-prepared for the identity and networking sections.

**Do I need AZ-104 before AZ-500?**
Not officially required, but strongly recommended. AZ-500 assumes you understand Azure networking, compute, and storage concepts.

**What career roles need AZ-500?**
Cloud security engineers, SOC analysts, Azure security architects, and compliance specialists.
